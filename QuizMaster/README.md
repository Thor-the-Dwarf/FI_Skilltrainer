# QuizMaster

Dieser Ordner ist der Arbeitsbereich für:

- Formeln zur Aufgabenbildung
- Zusammenfassungen der vorhandenen Begriffserklärungen
- Oberthemen- und Themen-Objekt-Strukturen
- SQLite-Arbeitsdatenbank
- interne Ableitungs- und Hilfsskripte

Nicht hierhin verschoben wurden die eigentlichen Kursdaten und Quizdateien, weil sie Nutzdaten bleiben.
